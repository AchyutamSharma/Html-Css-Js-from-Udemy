Methods of JavaScript :

  1. getAttribute() :

    2. The getAttribute() method returns the value of an element's attribute.

  2. setAttribute() :

    3. The setAttribute() method sets a new value to an attribute. If the attribute does not exist, it is created first.

  3. style property :

    4. The style property returns the values of an element's style attribute. The style property returns a CSSStyleDeclaration object. The CSSStyleDeclaration object contains all inline styles properties for the element. It does not contain any style properties set in the section or in any external style sheets.
